# Growth Track

**This folder is separate from the paid SHIP Method OS product.** It's a lead-generation / audience-building system used to promote the repo — not a repositioning of the repo itself, which stays commercial (see [`../LICENSE.md`](../LICENSE.md)).

Use this if you want to build an audience or email list around the SHIP Method before or alongside selling the core repo.

## Files

| File | What it's for |
|---|---|
| [`LANDING_PAGE_TH.md`](./LANDING_PAGE_TH.md) | Thai-language landing page (~3,000 words) — goal is email capture, not direct sale |
| [`EMAIL_SEQUENCE.md`](./EMAIL_SEQUENCE.md) | 6-email nurture sequence (Day 0/1/3/5/7/14) that softly introduces the paid repo by Day 14 |
| [`CONTENT_SYSTEM.md`](./CONTENT_SYSTEM.md) | Hook templates, content ideas, a 4-week calendar, and platform CTAs across TikTok/FB/LinkedIn/X/Threads/YouTube |
| [`SHIP_CANVAS.md`](./SHIP_CANVAS.md) | "The SHIP Canvas" — a 1-page lead magnet (Idea → Product Plan in 15 minutes), with format specs for PDF/Notion/FigJam/Miro |
| [`LEAD_CAPTURE_STRATEGY.md`](./LEAD_CAPTURE_STRATEGY.md) | The funnel design: Content → Landing Page → Email Capture → Repo → Sequence → Community |
| [`EXAMPLE-PROJECTS/`](./EXAMPLE-PROJECTS) | 6 short, shareable example briefs (booking, community, agency, SaaS, dashboard, internal tool) for use as teaser content |

## How These Connect

```
Content (CONTENT_SYSTEM.md)
        ↓
Landing Page (LANDING_PAGE_TH.md) — "⭐ Join SHIP Builder List"
        ↓
Email Capture
        ↓
Email Sequence (EMAIL_SEQUENCE.md) — Day 14 introduces the paid repo
        ↓
Community / Paid SHIP Method OS repo
```

The Growth Track exists to fill the top of that funnel. The paid repo (everything outside this folder) is what the funnel sells into — keep the two purposes distinct.
